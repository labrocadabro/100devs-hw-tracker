#100Devs Homework Tracker

This tracker is a little tool I built for myself because I didn't keep up with the homework as I watched the videos. It contains far less information than other resources because I wanted something that focused exclusively on homework. If you're in need of links to class videos, follow along materials, slides, etc, check out 100 Devs Follow Along or 100 Devs Course List, and huge credit goes to the author of this homework checklist which I used as my main source of information.

Checking the done checkbox will check off all the tasks for that week, and checking off all the tasks will automatically check done. hide completed will hide rows when their done checkbox is checked; it will not hide individual tasks. Done will be checked and classes will be hidden by hide completed even if the optional pushwork is not complete. The site uses localstorage to keep track of checkbox status, so as long as you keep using the same browser, your completions will be tracked.
